WHAT IS ARVI
------------
ARVI (Agentic Regeneration Via Intelligence) is an autonomous environmental
intelligence system. It deploys IoT sensor nodes in forests, rivers, and soil.
An AI agent reads every sensor reading, detects threats using LLM analysis,
logs events onchain, and triggers automatic USDC payments to node operators.

No human in the loop. The agent acts in under 2 seconds.

ARVI is NOT:
- A dashboard (dashboards don't act)
- An environmental NGO platform (not grant-dependent)
- A general monitoring tool (it executes, not just reports)

ARVI IS:
- An autonomous intelligence layer between environmental signals and action
- The 4th step in a research trajectory: REMBU → SEN → AONA → ARVI
- The first system that detects forest threats AND pays responders automatically

---

CORE LOOP (technical)
---------------------
1. Sensor node reads: temperature, humidity, CO2, soil moisture, AQI,
   biodiversity score (audio), pathogen risk (AI model), canopy density
2. POST /api/analyze { node_id }
3. lib/bankr.ts → Bankr LLM Gateway (gemini-2.5-flash) analyzes all fields
4. Returns: severity, alert_type, description, recommended_action, confidence
5. lib/locus.ts → triggers USDC payment to operator wallet
6. lib/agent-log.ts → appends to agent_log.json (onchain evidence)
